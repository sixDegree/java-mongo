
# java-mongo

## Use Mongo driver

1. Dependency Jar:
```xml
<dependency>
<groupId>org.mongodb</groupId>
<artifactId>mongodb-driver</artifactId>
<version>3.9.0</version>
</dependency>
```

2. Usage: Refer to MongoTest.java
